My UX / UI Skills

User Research


Studying target users and their requirements, to add realistic contexts and insights to design processes. I adopt various methods to uncover problems and design opportunities.

UI Design


Designing interfaces in software or computerised devices, focusing on looks or style. I aim to create interfaces which users find easy to use and pleasurable based on user research.

Prototyping


Building low-fidelity and high-fidelity prototypes to implement ideas in tangible forms from paper to digital to capture design concepts and test on users.

Responsive Web Design


Conceptualising websites from an atomic design structure starting with a mobile-first approach. I design for all viewport sizes to allow more accessibility for users.

Mobile Design & App


Approaching UX and UI design on mobile devices, with a keen interest. I design for both mobile apps and websites as more and more users today are accessing the web through mobile.

Storytelling


Capturing audience through storytelling is key to a UX UI design that stands out in the crowd. Storytelling speaks to the essence of the product and makes it memorable for users.
